# A Simple Mod adding Zyns to minecraft!
# Featues
- > Zyn Packs
  - Allow the user to store zyns in packs of 20.
- > Zyns
    - Give the user effects depending on the nicotine concentration:
      - 1 interval Haste 2
      - 2 interval Speed 2
      - 3 interval Strength 2
      - 7 interval Nausea
